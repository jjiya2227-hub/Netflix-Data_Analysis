# 🎬 Netflix Movies & TV Shows Analysis

## 📌 Project Overview
This project performs an Exploratory Data Analysis (EDA) on the Netflix Movies & TV Shows dataset using Python. It covers data cleaning, visualization, feature engineering, outlier detection, and business insights to understand Netflix's content library.

---

## 🎯 Objectives
- Understand the Netflix dataset
- Clean and preprocess the data
- Analyze movies and TV shows
- Explore content distribution by country, year, rating, and genre
- Perform feature engineering
- Detect outliers
- Generate business insights and recommendations

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

The project uses the **Netflix Movies & TV Shows** dataset (`netflix_titles.csv`).

## 💡 Key Insights
- Movies make up the majority of Netflix's catalog.
- The United States contributes the highest number of titles.
- Netflix content increased significantly after 2010.
- TV-MA is the most common content rating.
- International Movies are the most popular genre.
- Most movies have a duration between 80–120 minutes.
